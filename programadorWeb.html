programador Web

Um programador web é responsável por desenvolver, codificar, testar, manter e solucionar o código de sites, aplicações e portais para a internet. As suas principais atividades incluem: Estruturação de sites e aplicações Participação em reuniões Realização de testes Montagem, depuração e testes de programas Correções necessárias para atender às necessidades dos usuários Otimização do código para navegação O programador web pode atuar em duas áreas: front-end e back-end: Front-end Trata-se da parte gráfica do site, como o layout, menus e rodapés. O programador web deve preocupar-se com a experiência do utilizador. Back-end É responsável pelo servidor que hospeda o site, garantindo que os dados enviados ao navegador estão corretos. Existem também programadores web full-stack, que têm conhecimentos em ambas as áreas.
